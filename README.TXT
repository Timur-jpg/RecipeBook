PROJECT TITLE: RecipeBook
PURPOSE OF PROJECT: A digital recipe book application that allows users to manage, search, rate and scale recipes.
VERSION: 1.0
DATE: May 2026
HOW TO START THIS PROJECT:
  In BlueJ: Open project → right-click Main → Run
  Outside BlueJ: 
    1. Open terminal in project folder
    2. Type: javac *.java
    3. Type: java Main
AUTHORS: Timur Turdaliev
USER INSTRUCTIONS:
  Use the number menu to navigate.
  1. Print book details
  2. List all recipes
  3. List by type
  4. Search by title
  5. Search by ingredient
  6. Search by tag
  7. Print recipe details
  8. Rate a recipe
  9. List top rated recipes
  10. Scale a recipe
  0. Exit
